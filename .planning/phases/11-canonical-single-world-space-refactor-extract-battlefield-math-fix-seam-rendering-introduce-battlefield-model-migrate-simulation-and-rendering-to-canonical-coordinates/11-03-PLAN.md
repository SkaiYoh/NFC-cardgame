    **Step B: Create battlefield.c**

    Create `src/core/battlefield.c` with:

      ```c
      static float bow_offset(int lane, float t, float laneWidth) {
          if (lane == 1) return 0.0f;
          float bow = sinf(t * PI_F) * LANE_BOW_INTENSITY * laneWidth;
          return (lane == 0) ? -bow : bow;
      }
      ```

    - `battlefield_init`:
      1. Set `bf->boardWidth = BOARD_WIDTH`, `bf->boardHeight = BOARD_HEIGHT`, `bf->seamY = SEAM_Y`
      2. Initialize territories:
         - `bf->territories[SIDE_TOP].bounds = (Rectangle){0, 0, BOARD_WIDTH, SEAM_Y}`  (per D-02)
         - `bf->territories[SIDE_BOTTOM].bounds = (Rectangle){0, SEAM_Y, BOARD_WIDTH, BOARD_HEIGHT - SEAM_Y}` (per D-02)
         - For each territory: copy biome, biomeDef pointer, call `biome_copy_tiledefs`, set tileDefCount, call `biome_copy_detail_defs`, set detailDefCount, call `tilemap_create_biome` with the territory bounds
      3. Generate canonical lane waypoints for both sides:
         - For SIDE_BOTTOM (P1): spawnY = SEAM_Y + (BOARD_HEIGHT - SEAM_Y) * 0.8 = 960 + 768 = 1728. destY = 0 + SEAM_Y * 0.2 = 192. Waypoints[0] = slot spawn pos. Remaining waypoints: interpolate from spawnY toward destY with bow offset. Y values strictly DECREASE (per D-03).
         - For SIDE_TOP (P2): spawnY = 0 + SEAM_Y * 0.2 = 192. destY = SEAM_Y + (BOARD_HEIGHT - SEAM_Y) * 0.8 = 1728. Waypoints[0] = slot spawn pos. Remaining waypoints: interpolate from spawnY toward destY with bow offset. Y values strictly INCREASE (per D-04).
         - Total travel distance: |1728 - 192| = 1536px -- matches the current 2.0 depth units * 0.8 * 960 = 1536px (per Pitfall 2 in RESEARCH.md)
      4. Compute slot spawn positions for both sides:
         - SIDE_BOTTOM slot i: x = (i + 0.5f) * laneWidth, y = 1728
         - SIDE_TOP slot i: x = ((2 - i) + 0.5f) * laneWidth, y = 192 (mirror here because worldPos is the canonical position of slot i when rendered for P2)
      5. Zero out entities: `bf->entityCount = 0`, `bf->bases[0] = bf->bases[1] = NULL`

    - `battlefield_cleanup`:
      1. Free tilemaps: `tilemap_free(&bf->territories[SIDE_BOTTOM].tilemap)`, same for TOP
      2. Free entities: iterate `bf->entities`, call `entity_destroy`, set NULL
      3. Zero entityCount

    - `battlefield_add_entity`: same pattern as player_add_entity but using `MAX_ENTITIES * 2` limit
    - `battlefield_remove_entity`: swap-with-last pattern
    - `battlefield_find_entity`: linear scan by ID
    - `battlefield_territory`: return `&bf->territories[side]`
